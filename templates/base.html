외부 연동 데이터 API

이 키를 가진 외부 사이트가 아래 주소로 TRMT 데이터를 읽기 전용(JSON)으로 가져갈 수 있습니다. 출장 경비는 제외됩니다. 키는 외부에 노출되지 않도록 주의하세요.

GET /api/ext/all — 전체(선박·이슈·CS·Vetting·요약·DD·Boarding·일정·Class Status)
GET /api/ext/issues — Daily 업무
GET /api/ext/surveys — Condition Survey
GET /api/ext/vettings — Vetting
GET /api/ext/vetting-digests — Vetting 선박별 SIRE 요약(상태·항구·OBS·지적상세)
GET /api/ext/dock-reports — Dry Dock
GET /api/ext/boarding-reports — Boarding
GET /api/ext/calendar — 일정
GET /api/ext/summaries — 업무 요약(요약 탭)
GET /api/ext/class-status — 선급 Class Status
※ 모든 레코드에는 변하지 않는 고유키 ref(예: vetting:12, vt_finding:34, class_status:5)가 포함됩니다. 외부 사이트에서 특정 데이터를 식별·연동할 때 이 값을 주소처럼 사용하세요. 선박 연결은 vessel_key(선명 소문자)로 매칭 가능합니다. (이 키들은 API 응답에만 있고 사이트 화면에는 표시되지 않습니다.) 

    

      인증: 헤더 X-API-Key: 키 또는 쿼리 ?key=키. 선박 매칭은 vessel_key(선박명 소문자) 사용.
